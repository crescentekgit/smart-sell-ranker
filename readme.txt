SmartSell Ranker Plugin will automatically assign the top selling products to a category.

== Description ==

SmartSell Ranker Plugin will automatically assign the top selling products from a set time period to a set category. This assignment will be done through a scheduled Cron job that will perform product updates on a 24-hour basis with additional features to manage and optimize product categorization, enabling dynamic changes based on sales volume.

== Installation ==

MODERN WAY:

1. Go to the WordPress Dashboard “Add New Plugin” section.

2. Search For “SmartSell Ranker”.

3. Install, then Activate it.

OLD WAY:

1. Upload the plugin folder to the /wp-content/plugins/ directory.

2. Activate the plugin through the ‘Plugins’ menu in WordPress.
